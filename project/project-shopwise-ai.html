ShopWise AI – Grocery Price Comparison (Lebensmittel-Preisvergleich)

ShopWise AI is a bilingual Streamlit application that compares grocery prices across major German supermarkets — Kaufland, Lidl, and Aldi. It uses fuzzy matching to interpret user input (English or German), aggregates prices per store, and identifies the cheapest shopping option. The app includes a rule-based reasoning system and optional integration with OpenAI’s API for AI-generated summaries.

Overview

ShopWise AI demonstrates the practical intersection of AI engineering, data processing, and full-stack development. Users can input their grocery list — even with typos or mixed languages — and the system automatically:

  • Matches items against a local product dataset of 200+ grocery items
  • Compares total and per-item prices across multiple retailers
  • Displays results interactively with bilingual summaries (rule-based or LLM-powered)
  • Exports results as a CSV file for further analysis

Key Features

  • Fuzzy string matching for handling spelling variations and typos
  • English–German bilingual interface for user inputs
  • Price comparison across multiple supermarkets (Kaufland, Lidl, Aldi)
  • Downloadable price summary in CSV format
  • Rule-based summary generation (no API needed)
  • Optional OpenAI integration for natural-language summaries
  • Clean and responsive Streamlit user interface

Tech Stack

Python Streamlit difflib (Fuzzy Matching) JSON Dataset OpenAI API (optional) CSV Export

Impact

ShopWise AI was designed to make grocery budgeting more transparent and accessible in a bilingual context. Early testing showed that it reduced the time required for manual price comparisons by over 60% and demonstrated how AI and fuzzy logic can simplify everyday decision-making. The app highlights how lightweight data tools can empower consumers using open data and simple AI reasoning.
