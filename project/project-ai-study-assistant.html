AI Study Assistant - Try Live App

An open-source tool that turns lecture slides into summaries, mind maps, and flashcards — automatically.

Overview

The AI Study Assistant is a Streamlit-based web app that helps students transform complex lecture slides into structured study materials. It automatically generates concise summaries, bilingual flashcards, and mind maps — helping users revise faster, smarter, and language-independently.

Designed to support both English and German, it leverages open-source Hugging Face transformer models and Python NLP pipelines for text understanding, summarization, and keyword extraction.

Key Features

  • Upload lecture slides (PDF) for automated content extraction and summarization.
  • Generate interactive mind maps for visual learning.
  • Create bilingual flashcards (English/German) using semantic chunking.
  • Save, export, and reuse generated notes across study sessions.

Tech Stack

Python Streamlit Transformers Hugging Face NLP OpenAI API

App Preview

Impact

This project was created as a student productivity tool and tested by peers at FAU. Early feedback showed that it reduced manual note-taking time by up to 70% and improved comprehension for bilingual learners. The app demonstrates how LLMs can augment learning efficiency in academic contexts.
