MediScript AI

MediScript AI is a general handwriting recognition system built for handwritten notes, prescriptions, essays, and single-word OCR. It uses PaddleOCR for text detection, TrOCR for handwriting recognition, and RapidFuzz for optional medicine correction in prescription mode. The application is deployed with Streamlit and designed as a practical, user-facing OCR tool.

Python Streamlit PaddleOCR TrOCR RapidFuzz OCR

Overview

This project was designed to solve a real OCR problem: extracting text from handwritten documents in a simple and usable way. Unlike standard OCR tools that work best on printed text, MediScript AI focuses on handwritten input and supports three practical modes: general OCR, prescription OCR, and single-word recognition.

  • Detects text regions from handwritten images
  • Recognizes handwriting using transformer-based OCR
  • Supports optional medicine correction for prescription input
  • Provides a simple web-based Streamlit interface
  • Can be extended further in Version 2 for improved accuracy

Core Features

  • General OCR for handwritten documents
  • Prescription OCR with optional medicine-name correction
  • Single word recognition with candidate predictions
  • Support for PaddleOCR and TrOCR backends
  • Cloud deployment with Streamlit Community Cloud
  • Portfolio-ready end-to-end AI application

System Architecture

Image Input ↓ PaddleOCR Text Detection (bounding boxes) ↓ Crop Regions ↓ TrOCR Handwriting Recognition ↓ RapidFuzz Optional Medicine Correction ↓ Extracted Text Output

Tech Stack

Python Streamlit PaddleOCR PaddlePaddle Transformers TrOCR RapidFuzz Pillow NumPy Pandas
